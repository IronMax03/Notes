# Discrete Mathematics Notes

Concise, course-oriented notes for Discrete Mathematics from Webster University ([Discrete_Mathematics.tex](Discrete_Mathematics.tex) and [Discrete_Mathematics.pdf](Discrete_Mathematics.pdf)).
- Class: MATH 2410 (from the document header)
- Format: Compact 2-column PDF built with LaTeX
- Scope: sets, functions, counting, logic, proofs, graphs.

## Additional Sheets
- CSCI 2824 Discrete Structures (wich is an equivalent classe from the University Of Colorado Boulder): [Discrete_Structures_CUB.tex](Discrete_Structures_CUB.tex) and [Discrete_Structures_CUB.pdf](Discrete_Structures_CUB.pdf) are included in this folder cheat sheet.

- Counting: additive/multiplicative principles, permutations, combinations, binomial coefficient, binomial theorem
